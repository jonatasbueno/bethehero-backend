# Be the hero - Backend
Backend da aplicação Be The Hero


**Tecnologias utilizadas**

- Express
- Knex
- Sqlite3
- Nodemon
- Cors

**Explorados**
- Migrations através do Knex


Para rodar a aplicação é necessário um ambiente possuindo Node 12 e NPM 6

Intalando dependências:

```
npm install
```

Subindo backend

```
npm run start
```
